# Code of Conduct

## Our Pledge

OpenPlayground is committed to providing a welcoming, inclusive, and harassment-free experience for everyone, regardless of experience level, gender, gender identity, sexual orientation, disability, personal appearance, race, ethnicity, age, religion, or nationality.

We aim to foster a positive environment where developers can learn, build, share ideas, and collaborate respectfully.

## Scope

This Code of Conduct applies to all project spaces, including but not limited to:
- GitHub repositories
- Issues and pull requests
- Discussions and comments
- Community interactions related to OpenPlayground

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.  
They have the right to remove, edit, or reject contributions, comments, or pull requests that violate these guidelines.

Repeated or severe violations may result in:
- Temporary or permanent bans from the project
- Removal of contributions
- Restriction from community participation
